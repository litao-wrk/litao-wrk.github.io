Generalized linear model(GLM)

Ordinary linear regression

  • linear-response model
    • normal distribution of response variable
      • response variable 和 error是同类型的分布
        • E = R - w*x
    • link function of response y’ = f(y) = y
    • 有什么问题？
      • 如果Y不是正态分布会怎么样？
      • Y和X是线性关系
      • overdispersion
      • homogeneity of variance
  • GLM
    • Model assumptions
    • distribution of response variable(exponential family)
      • bernoulli()
        • log-odds(logit)
        • logit
          • quantile function of logistic distribution
          • quantile function is inverses of the cumulative distribution function (CDF) of a probability distribution
      • binomial distribution is the sum of independent and identically distributed Bernoulli random variables.
        • X =X1+⋯+Xn
      • poisson
    • link function of response variable(arbitrary)
      • logit = link(y)
    • predictor(Systematic Component) => link => response(Random Component)
    • maximum likelihood estimation(MLE)
      • large-sample approximations
      • minimize neg-likelihood
        • (maximize a posterior) MAP: bayes
          • MLE = MAP with uniform prior
    • 模型评估
      • 参数解释
      • Model fit: R^2^, residual analysis, F-statistic
    • 问题？
      • 线性模型
      • Responses must be independent
  • categorical(nominal)
    • ordinal
    • numerical(equally spaced ordinal)
